Vim命令记忆方法

Posted on 2016-03-12

模式

  • i → 切换到插入模式
  • ESC → 切换到命令行模式

打开/保存/退出/改变文件(Buffer)

  • :e → 打开一个文件
  • :w → 存盘（write）
  • :saveas → 另存为
  • :wq → 保存并退出（write and quit）
  • :q! → 退出不保存 （quit）
  • :qa! → 强行不保存退出所有的正在编辑的文件（quit all）
  • :bn 和 :bp → 切换打开的多个文件（next previous）
  • :n → 切换到下一个文件（next）

移动光标

  • gg → 移到第一行开头
  • G → 移动到最后一行的开头
  • 0 → 移动到光标所在行的行头
  • $ → 移动到光标所在行的行尾
  • ^ → 移动到本行第一个不是blank字符的位置
  • g_ → 移动到本行最后一个不是blank字符的位置
  • nG → 移动到第 n行的行头
  • w → 到下一个单词的开头（word）
  • e → 到下一个单词的结尾（end）
  • b → 到上一个单词的开头（begin）
  • % → 匹配括号移动
  • /pattern → 搜索pattern的字符串（如果搜索出多个匹配，可按n键到下一个）

插入/删除

  • a → 在光标后插入（after）
  • o → 在当前行后插入一个新行
  • O → 在当前行前插入一个新行
  • cw → 替换从光标所在位置后到一个单词结尾的字符
  • dd → 删除光标所在行（delete delete）

拷贝/粘贴

  • yy → 拷贝当前行
  • p → 当前位置之后粘贴（paste）
  • P → 当前位置之前粘贴（paste）

Undo/Redo

  • u → undo
  • Ctrl + r → redo
